# Futsal Manager

15人でのフットサルの試合管理・MVP選出を行うWebアプリです。  
Google Apps Script（GAS）で完結し、スマホブラウザから利用できます。

## データ構造

全てのデータは**イベント単位**で管理されます。イベントを横断した集計はありません。

```
イベント
├── メンバー（このイベントの参加者）
├── ラウンド（第1試合、第2試合...）
│   ├── チーム分け
│   ├── スコア
│   └── 得点者
├── アンケート（Googleフォーム）
├── MVP結果
└── イベント内成績
```

## 機能

### イベント実施前
- **メンバー登録** - 名前、年次（社会人歴）、サッカー経験の有無、幹事フラグ
- 個別追加・一括追加（改行区切り）に対応

### 当日（フットサル中）
- **チーム分け** - AI自動分け（経験・年次を考慮）またはランダム分け
- **複数ラウンド** - 第1試合、第2試合...と何回でも追加可能
- **試合結果管理** - 得点者ボタンでスコア自動計算

### 試合後
- **アンケート** - Googleフォームを自動生成。各メンバーへのコメントを収集
- **MVP選出** - 定量評価 + 定性評価でAIが選出。MVP/準MVPの人数は設定可能
- **イベント内成績** - 得点数・勝敗を集計

## セットアップ手順

### 1. Google スプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) で新しいスプレッドシートを作成
2. 「拡張機能」→「Apps Script」を開く

### 2. ファイルをコピー

Apps Script エディタで以下のファイルを作成し、`src/` フォルダ内の内容をコピー：

| Apps Script ファイル名 | コピー元 |
|---|---|
| `Code.gs` | `src/Code.gs` |
| `Members.gs` | `src/Members.gs` |
| `Events.gs` | `src/Events.gs` |
| `Rounds.gs` | `src/Rounds.gs` |
| `Survey.gs` | `src/Survey.gs` |
| `Mvp.gs` | `src/Mvp.gs` |
| `index.html` | `src/index.html` |
| `css.html` | `src/css.html` |
| `js.html` | `src/js.html` |

### 3. シートを初期化

1. `Code.gs` を開く
2. 関数セレクタで `initializeSheets` を選択 → ▶ 実行
3. 初回は権限の承認が求められるので許可

### 4. Webアプリとしてデプロイ

1. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」
2. 次のユーザーとして実行：**自分**
3. アクセスできるユーザー：**全員**
4. URLをコピーして友達に共有

## 使い方

1. **イベント作成** → 日付と名称を入力
2. **メンバー登録** → 15人の名前・年次・経験を登録（一括追加可）
3. **チーム分け** → AI分け or ランダム → 試合開始
4. **試合記録** → 得点者をタップ（スコア自動計算）
5. **次の試合** → 新しいラウンドを追加して繰り返し
6. **アンケート** → フォーム自動生成 → URL共有 → 回答取得
7. **MVP選出** → AI判定 → 結果発表

## 技術構成

- **バックエンド**: Google Apps Script（6ファイル）
- **フロントエンド**: HTML + CSS + JavaScript
- **データ**: Google スプレッドシート（7シート）
- **アンケート**: Google フォーム（自動生成）
